Настройка роутера

Как прописать шлюз в DHCP-настройках роутера для всей сети

Что прописать в роутер
Default Gateway (основной шлюз) —
Primary DNS (первичный DNS) —
Адрес веб-интерфейса роутера —
ℹ️ Оба поля (Gateway и DNS) должны содержать один и тот же IP-адрес шлюза. После сохранения переподключите устройства к Wi-Fi.
Пошаговая инструкция
1

Откройте веб-интерфейс роутера

В браузере на любом устройстве откройте адрес роутера. Обычно это 192.168.1.1 или 192.168.0.1 (зависит от модели).

http://192.168.1.1
💡 Если не знаете адрес роутера — на вашем компьютере в сетевых настройках посмотрите поле «Основной шлюз» / «Default Gateway».
2

Войдите в настройки роутера

Введите логин и пароль роутера. Стандартные значения: admin / admin или admin / пусто. Обычно написаны на наклейке снизу устройства.

3

Найдите раздел настроек DHCP

Раздел может называться по-разному в зависимости от производителя:

LAN → DHCP Server Сеть → DHCP Advanced → LAN Setup Network → LAN Home Network → DHCP
4

Установите Default Gateway

Найдите поле Default Gateway (или «Основной шлюз», «Router») и введите:

192.168.50.2
5

Установите Primary DNS

Найдите поле Primary DNS (или «Предпочитаемый DNS», «DNS Server 1») и введите тот же IP:

192.168.50.2
⚠️ Secondary DNS можно оставить пустым или указать 9.9.9.9 как запасной на случай недоступности шлюза.
6

Сохраните и примените настройки

Нажмите Save / Apply / Сохранить. Роутер может перезагрузиться — подождите 30–60 секунд.

7

Переподключите устройства

Устройства получат новые параметры сети (шлюз и DNS) при следующем обновлении DHCP-аренды. Для немедленного применения:

📱 Телефоны/планшеты: отключитесь от Wi-Fi и подключитесь снова 💻 Windows: ipconfig /renew в командной строке 🍎 macOS: выключите Wi-Fi на 5 секунд, затем включите 🐧 Linux: sudo dhclient -r && sudo dhclient

✓ Как проверить

После переподключения откройте любой заблокированный сайт (YouTube, Instagram и т.д.). Если работает — шлюз настроен верно.

Текущий внешний IP можно узнать на: api.ipify.org или 2ip.ru
